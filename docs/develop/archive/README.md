## 為什麼歸檔？

這些文件記錄了重要的技術決策過程和實驗結果，具有參考價值。但對於想快速了解**當前專案狀態**的人來說，這些歷史細節可能造成干擾。

因此我們將這些文件歸檔至此，保持主要文檔的簡潔性，同時保留歷史記錄供需要時查閱。
