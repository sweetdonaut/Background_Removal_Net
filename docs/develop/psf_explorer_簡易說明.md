# PSF Explorer 改進說明（簡易版）

## 給沒有光學/物理背景的讀者

本文用比喻和直覺說明這個工具改了什麼、為什麼要改。如果你想看完整數學請看技術版。

---

## 先解釋三個關鍵字

### 什麼是 PSF？

當光學儀器（顯微鏡、相機）拍攝一個「無限小的點」時，因為光的繞射本質，這個點在影像上不會是一個完美的小點，而會「散開」變成一個模糊圖案。

這個模糊圖案就叫 **PSF（Point Spread Function，點擴散函數）**。

換個方式想像：
- 真實世界中有一顆很小的星星
- 望遠鏡拍到的影像不會是 1 像素的點，而是會有一個小光暈
- 這個小光暈的形狀就是 PSF

PSF 形狀完全由「儀器本身」決定 ── 鏡頭多大、光圈是什麼形狀、有沒有像差，都會改變 PSF。

### 為什麼我們要關心 PSF？

我們在訓練 AI 偵測「亮點 defect」時，需要餵很多「假的 defect」給 AI 學習。這些假 defect 在影像上長什麼樣？**就是 PSF 的形狀**。

如果合成的 defect 跟真實儀器上看到的差太多，AI 上線後就會表現很差。所以我們必須**精確控制合成 defect 的形狀**，這需要一個能模擬真實光學系統的 PSF 工具。

### PSF Explorer 是什麼？

一個網頁工具，拉幾個 slider 就能即時看到「不同光學設定會產生什麼 PSF 形狀」。我們用它來：

1. 視覺化試驗不同參數的效果
2. 挑出符合真實儀器的設定
3. 把選好的數字記錄到 yaml 設定檔，給訓練 pipeline 用

## 第一次改進：更多種光圈遮擋

### 原本只能模擬什麼？

最初版的 PSF Explorer 假設光圈是一個簡單的「環形」（甜甜圈形）。這個假設適用於最簡單的望遠鏡。

### 真實世界更複雜

實際的儀器光圈常常被機構結構阻擋。例如：

- **望遠鏡有支架**：鏡片中央有十字形的支架，會在 PSF 上產生明顯的「十字繞射光芒」（你拍夜空看到的星星「閃光」就是這原因）
- **顯微鏡有方形光闌**：光圈中央有一個方形阻擋
- **狹縫光闌**：只保留中央一條水平或垂直的窄帶

每種遮擋都會在 PSF 上產生**不同的形狀特徵**。如果真實儀器有遮擋，模擬時不算進去，合成的 defect 跟真實的就會「形狀對不起來」。

### 我們新增了 5 種遮擋設定

| 設定 | 模擬什麼情況 |
|------|-----------|
| 中央方形遮擋 | 光圈中央有方形 stop |
| 水平遮擋條 | 像支架的橫條把光圈攔腰切斷 |
| 垂直遮擋條 | 直條阻擋 |
| 上下外側裁切 | 只保留中央水平帶 |
| 左右外側裁切 | 只保留中央垂直帶 |

這次改進**不涉及新的物理理論**，只是讓「光圈長相」可以更接近真實儀器。

## 第二次改進：高 NA 透鏡的物理模型

這次是更深層的改進，加入了一個叫做 **Richards-Wolf 向量繞射** 的計算方法。

### NA 是什麼？

NA（Numerical Aperture，數值孔徑）可以理解為「**透鏡能多『敞開』地收光**」。

打個比方，想像你拿著吸管喝水：
- **細吸管**（NA 小）= 只能從正前方一個小角度收水
- **超大喇叭口的吸管**（NA 大）= 能從很大的範圍把水捲進來

NA 越大，鏡頭能收的光越多、解析度越高。但也讓物理計算變得**完全不同**。

### 為什麼舊模型在高 NA 不準？

舊模型假設「光線從正前方進來」（像細吸管）。這假設在 NA < 0.5 還算 OK。但我們的實機 **NA ≈ 0.95**（接近物理上限），光線是從**很大的角度斜著進來**。

當光斜著進來，幾件事會發生：

1. **PSF 形狀依「光的偏振方向」而變**（舊模型完全沒這概念）
2. **光的「上下方向震動分量」會出現**（這個就是 Ez 分量）
3. **PSF 會被沿著偏振方向「擠扁」**

舊模型把光當成普通水波（純量）；新模型把光當成電磁波（向量），有方向性。

### 偏振是什麼？

光除了「亮度」之外，還有一個叫「偏振」的性質，描述「電場震動的方向」。

打比方：
- 把繩子上下擺動 = 線偏振
- 把繩子轉圈搖 = 圓偏振
- 不同偏振光聚焦時形狀**真的不一樣**（這不是錯覺，是電磁學的真實效應）

新版加了 7 種偏振選擇：

| 偏振 | 像什麼 |
|------|-------|
| 線偏振 X / Y / 45° | 把繩子往單一方向擺 |
| 圓偏振 R / L | 把繩子轉圈搖（左旋或右旋）|
| 徑向偏振 | 從中心往四面八方放射狀震動 |
| 角向偏振 | 沿著圓周圍繞震動 |

### 各偏振 PSF 的差異（一句話描述）

- **線偏振 X**：PSF 變成垂直橢圓（沿 X 被壓縮）
- **線偏振 Y**：跟 linX 正交，變水平橢圓
- **圓偏振**：對稱圓形（最像舊模型結果）
- **徑向偏振**：PSF 中心特別亮，是超解析顯微鏡常用的招數
- **角向偏振**：⚠️ **甜甜圈形（中心是空的）**，**目前不建議用**，因為產生的 defect 形狀很奇怪，需要更多驗證

### 實際看效果

打開 PSF Explorer 網頁，試著：
1. 把所有像差歸零，調出乾淨的 PSF
2. 開啟「Richards-Wolf 向量模式」
3. 切換偏振 linX / linY / radial，**會看到 PSF 形狀明顯不同**

這就是新模型補回來的物理。

### 為什麼這次改進是必要的？

我們的儀器 NA = 0.95。用舊（純量）模型算出來的 defect 跟真實看到的 defect 形狀差距很大。如果 AI 是用舊模型的 defect 訓練的，上線後會 fail。

**新模型才能讓 AI 看到「跟真實儀器一樣的 defect」。**

## 副作用：能不能繼續用舊版方式？

可以。新版**完全向下相容**：
- 不開「向量模式」開關 → 行為跟原版完全一樣
- 既有的 yaml 設定檔（type1.yaml, type2.yaml, ...）不用改

向量模式是「opt-in」（自願選用）的功能。
